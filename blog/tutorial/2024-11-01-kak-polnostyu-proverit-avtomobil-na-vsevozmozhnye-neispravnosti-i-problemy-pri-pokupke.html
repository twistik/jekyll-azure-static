Launch x431 is a professional, multi-marochrome automobiles scanner designed for multifaceted diagnostics of electronic blocks of most representatives of European, Asian and North American automobilesprom. Their number reaches eighty, including Japanese right-hand drive cars. In conjunction with Scanmatic 2, serves domestic automobiles. It works on Android platform, all functions of android tablets and smartphones are available.

The test itself covers almost all elements of the automobilesmobile control unit, equipped with sensors. It draws current from the automobiles' mains supply, ranging from 5 to 35 volts

What the scanner can do:

Identification - identifying the type of electronic block required to select the correct protocol (from a list in the scanner), allowing you to optimise diagnostics and get the correct results.

Reading error codes. Faults in various systems, assemblies and parts, as signalled by sensors, are stored as codes in the automobilesmobile ECU, from where they are displayed on the screen of a connected tablet, in decoded form.

Freeze Frame - additional data to find out the common causes of the malfunction. That is, it takes readings from the systems and units of the automobiles that interact with the faulty component. This helps to optimise diagnostics.

Removing faults from memory. After the fault is cleared, the corresponding code is erased from memory by the X-431. The code may be erroneously recorded (e.g. dust or moisture on the sensor). In this case, once erased, it will not occur again, unlike a real failure.

Data Stream, Live Data, Measuring values - output of sensor signals entering the ECU. In this case, the operator has the opportunity to identify faults that the ECU has not recognised.

External Control. Launch X-431 is able to temporarily block the control functions of the automobiles ECU, allowing the operator to ‘manually’ start or change the parameters of the mechanisms.

.

There is also a special functionality (you can only use it if you know what you are doing!):

Adaptation. It is used to pair different components with each other. For example: blocks and peripherals after part replacement, immobilisers and ECU and so on.

Coding - making changes to the control. For example, displaying additional information on the panel or removing unused functions.

Programming - flashing the memory (albeit with limited capabilities).Coding - making changes to the control unit, such as displaying additional information on the panel or removing unused functions.

How to use the scanner:

Connect to the machine's ECU via the connector under the automobiles interior panel. Set up a wireless connection and register. Create an account, customise the basic settings to suit your needs

When you switch on the device, by default, the programme is in diagnostic mode. You need to select from the list the engine of the model under test by its marking. After diagnostic operations are performed, the results are displayed on the screen in the form of error code decoding. All information about previous diagnostic sessions (reports, time, location, and more) is stored in the Accounting tab.

When outputting test results, the scanner uses both standard diagnostic software and can create screenshots of the screen to send to a printout or flash drive. It can also send to print via WiFi.
